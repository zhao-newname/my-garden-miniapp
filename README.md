# Garden Management Mini Program

A WeChat Mini Program for managing home gardening.

## Features

- Plant Management
- Watering Reminders
- Planting Guides

## Development Environment

- WeChat Developer Tools
- Node.js
- Cloud Development

## How to Run

1. Clone the project
2. Import to WeChat Developer Tools
3. Configure cloud environment
4. Start development
